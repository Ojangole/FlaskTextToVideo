Application: A text to video generator using Stable Diffusion API.

To run, 
- create a virtual environment and set it up. (https://flask.palletsprojects.com/en/2.2.x/installation/)
- run "pip install replicate" in the virtual environment terminal.
- dont write the code in the "venv" folder
- in the virtual environment terminal, type "FLASK_APP=gen.py flask run"

generator.py is implementation of the text to video app without flask. The flask implementation is in TextGen2.
